List

In HTML, lists are used to group related items, making content more organized and readable. There are three main types:

Ordered List
Displays items in a numbered format, typically used when order matters, like steps in a process.
Unordered List
Presents items in a bulleted format, suited for lists without a specific order, such as categories or features.
Definition List
Contains pairs of terms and descriptions, useful for glossaries or FAQs. Each term is followed by a definition.

In CSS, lists can be customized extensively. You can change bullet styles with list-style-type (e.g., circle, square, decimal), or remove bullets using list-style: none;. Custom icons or images can replace default bullets with list-style-image. The appearance of lists can also be modified with properties like padding, margin, and color. For nested lists, indentation can be controlled by adjusting margin or padding values. You can also style lists horizontally by setting display: inline; on li elements, often used for navigation menus. Overall, HTML and CSS provide flexible options for creating and styling lists, making them versatile for various webpage layouts and designs.

Linking

In HTML, a link is created using the a (anchor) tag, allowing users to navigate between web pages or jump to different sections within a page. The `href` attribute within the anchor tag specifies the destination URL, which can be an external website or an internal link to another page on the same site. Additional attributes like `target="_blank"` open links in new tabs, while `rel="nofollow"` can instruct search engines not to follow the link, useful for SEO purposes. CSS provides control over how links look and behave. Links have default states like normal (unvisited), `:hover` (when the user hovers over them), `:active` (when the link is clicked), and `:visited` (for previously visited links). Using these pseudo-classes, developers can change link color, style, or add effects like underline or text shadow, enhancing user interaction and improving accessibility. For example, applying a color transition effect on the `:hover` state can create a smooth color change when the link is hovered, improving the user experience. Customizing link appearance with CSS is important for creating a cohesive, visually appealing design that aligns with the rest of the webpage styling and guides users effectively through the site.

Anchor Tag
Used to create links in HTML. The href attribute in the a tag defines the URL of the page the link goes to.
href attribute
An attribute in the a tag that specifies the destination URL or path of the link.
target attribute
Specifies where to open the linked document. Common values include:
  • _self (default): Opens in the same tab.
  • _blank: Opens in a new tab.
  • _parent: Opens in the parent frame.
  • _top: Opens in the full body of the window.
CSS Pseudo-classes for Link
Special states of links in CSS.
  • :link: Targets unvisited links.
  • :visited: Targets links that have already been clicked
  • :hover: Applies styles when the link is hovered over.
  • :hover: Applies styles when the link is hovered over.
Anchor ID
A method for linking to specific sections within a webpage by using an id attribute on an element. Links using #id scroll the page to the element with the matching id.

Selectors

A CSS selector is the first part of a CSS Rule. It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them. The element or elements which are selected by the selector are referred to as the subject of the selector.In other articles you may have met some different selectors, and learned that there are selectors that target the document in different ways â€” for example by selecting an element such as h1, or a class such as .special.In CSS, selectors are defined in the CSS Selectors specification; like any other part of CSS they need to have support in browsers for them to work. The majority of selectors that you will come across are defined in the Level 3 Selectors specification and Level 4 Selectors specification, which are both mature specifications, therefore you will find excellent browser support for these selectors.

Types Of Selectors :

Simple selectors
select elements based on name, id, class
Combinator selectors
select elements based on a specific relationship between them
Pseudo-class selectors
select elements based on a certain state
Pseudo-elements selectors
select and style a part of an element
Attribute selectors
select elements based on an attribute or attribute value

CSS Formatting

CSS (Cascading Style Sheets) formatting allows web developers to control the look and feel of a website, separating content from presentation. CSS formatting covers a range of stylistic adjustments, including layout, color, typography, spacing, and more, applied to HTML elements.

Types of CSS :

Inline CSS
An inline CSS is used to apply a unique style to a single HTML element.
Internal CSS
An internal CSS is used to define a style for a single HTML page.
External CSS
An external style sheet is used to define the style for many HTML pages
